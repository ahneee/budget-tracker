# Budget Tracker

A personal budget tracker web app built with Vite and TypeScript. Track income and expenses, set category budgets, and view monthly summaries — all stored locally in your browser.

**Budget Tracker: v1.0**

## Features

- Add **income** and **expense** transactions with description, amount, category, and date
- **Monthly dashboard** with net balance, income, expenses, and transaction count
- **Category budgets** with progress bars when you exceed limits
- **Analytics** with trends, insights, and category breakdown
- **Settings** for theme, currency, language, and profile
- **Export / import** your data as JSON
- Data persists in **localStorage** on your device

## Getting started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

The built files are in the `dist` folder — deploy them to any static host (e.g. [Vercel](https://vercel.com)).

## Tech stack

- [Vite](https://vite.dev/) — dev server and bundler
- TypeScript — type-safe app logic
- Vanilla DOM — no framework dependencies
